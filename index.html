Architecture

0 · Edit

Crop, rotate, flip, brightness / contrast / sharpen before sending to OCR

1 · Preprocess

Bilateral denoise → safe BG removal → CLAHE → Sauvola binarize → large blob removal → Hough skew fix

2 · CC Lines

Connected component Y-centroid clustering (25 px gap) finds line bands even when ink physically touches between lines

3 · VPP Words

Vertical Projection Profile per line band, top-20% blanked (shirorekha suppression) → inter-word gaps visible

4 · TrOCR

Fine-tuned model (20 epochs, RTX 4050, ~9% CER). Each word crop: 64px height, CLAHE enhanced, beam=5

5 · NLP + TTS

NFC normalise → Levenshtein spell fix → optional gTTS Nepali voice output
